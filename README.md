### Call Taxi Booking Application

This is a simple Java console application to simulate a Call Taxi Booking System.

## About the Project

Customers can book taxis between points (A to F).
The system allocates taxis based on:

Taxi availability

Nearest taxi to pickup point

Taxi with lower earnings (if multiple taxis are free)

If no taxi is available, the booking is rejected.

## Assumptions

4 Taxis (All start at point A)

Points: A, B, C, D, E, F

15 km between each point

1 hour travel time between points

Fare:

Rs.100 for first 5 km

Rs.10 per km after 5 km
